Todos los formularios en la web utilizan métodos para el envío de las distintas peticiones que realiza el usuario. Dentro de la arquitectura cliente-servidor, estas peticiones se gestionan mediante el protocolo HTTP y, actualmente, con mayor seguridad, a través de HTTPS. Cada solicitud debe ser atendida en un tiempo de respuesta adecuado conforme al modelo TCP/IP. Los principales métodos empleados en los formularios son:

  • GET: Este método se encarga de enviar la información del formulario por medio de la URL...
  • POST: Este método se encarga de enviar la información para registrar un cambio...
  • DELETE: Sirve para eliminar un recurso específico dentro del servidor.
  • PUT: Sirve para almacenar la información de un recurso y actualizarlo.
  • HEAD: Sirve para regresar el código de cabecera HTTP asociado a un recurso.
